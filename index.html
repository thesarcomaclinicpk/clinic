Backup & data

Everything is saved automatically in this browser on this computer. The backup file restores all patients, visits, medicines, documents and settings. X-ray images are kept in a separate large store — use “Export images” below to back those up too.

Auto-backup keeps one backup file continuously up to date — every save silently rewrites it. Choose a file inside your Google Drive or iCloud folder and your data is cloud-backed with zero effort. (Chrome may ask once per session to allow access — one click.)

With a password set, backup files (manual and auto) are AES-encrypted — safe even if your Google Drive is compromised. You'll be asked for this password when restoring. Write it down: a lost password means the backup cannot be opened by anyone.

X-ray images are stored separately from records — export them here to move or safeguard them.
